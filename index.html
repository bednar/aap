    Apiary Configuration

    For export to Apiary Blueprint add <apiary>...</apiary> to <configuration/>.

    Maven Plugin 

    <plugin>
    <groupId>com.github.bednar</groupId>
    <artefactId>aap</artefactId>
    <version>0.1-SNAPSHOT</version>
    <configuration>
        ...
    </configuration>
</plugin>
