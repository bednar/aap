aap:apiary

Full name:

com.github.bednar:aap:0.1-SNAPSHOT:apiary

Description:

Generate Apiary Blueprint.

Attributes:

  • Requires a Maven project to be executed.

Optional Parameters

Name Type Since Description
apiBaseURL String - API Base URL
Default value is: http://demoaap.apiary.io.
apiaryOutput File - Documentation output
Default value is: ${project.build.directory}/generated/apiary.
appDescription File - File with Description (support Markdown) of Application
Default value is: ${project.basedir}/description.md.
appName String - Name of Application
Default value is: Demo Application.

Parameter Details

apiBaseURL:

API Base URL
  • Type: java.lang.String
  • Required: No
  • Default: http://demoaap.apiary.io

apiaryOutput:

Documentation output
  • Type: java.io.File
  • Required: No
  • Default: ${project.build.directory}/generated/apiary

File with Description (support Markdown) of Application
  • Type: java.io.File
  • Required: No
  • Default: ${project.basedir}/description.md

appName:

Name of Application
  • Type: java.lang.String
  • Required: No
  • Default: Demo Application
