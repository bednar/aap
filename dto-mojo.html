aap:dto

Full name:

com.github.bednar:aap:0.1-SNAPSHOT:dto

Description:

Generate Data Transfer Object (DTO) from ApiModel.

Attributes:

  • Requires a Maven project to be executed.

Optional Parameters

Name Type Since Description
addTestSources Boolean 0.1 If Boolean.TRUE than use test sources path for search ApiModel.
Default value is: false.
dtoOutput File 0.1 DTO classes output directory.
Default value is: ${project.build.directory}/generated/dto.

Parameter Details

addTestSources:

  • Type: java.lang.Boolean
  • Since: 0.1
  • Required: No
  • Default: false

dtoOutput:

DTO classes output directory.
  • Type: java.io.File
  • Since: 0.1
  • Required: No
  • Default: ${project.build.directory}/generated/dto
